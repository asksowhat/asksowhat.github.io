 33. 对于表只能有惟一的主键码，但可有任意数量的UNIQUE属性或属性集。（  ）

45. 数据在语法上的约束称为完整性约束。（  ）

47. 每一个关系都应有一个主键，用来唯一标识一个元组。它的值不能为空，否则无法区分和识别元组，这就是域完整性约束。（  ）
48. 凡是属于3NF的必然是属于BCNF。（  ）
49. 从数据库逻辑设计来看，减少连接运算可以改善数据库的性能。（  ）
